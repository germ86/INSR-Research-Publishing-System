\section{Package Overview}

The INSR LaTeX package provides a LuaLaTeX-first document system for interdisciplinary trauma research. It includes shared style definitions, scientific-paper, presentation and clinical-manual classes, Overleaf-compatible build settings, and GitHub CI integration.

\section{Class Options}

\begin{description}
  \item[blindreview] Masks author and institution metadata where supported.
  \item[twocolumn] Enables a compact two-column article layout for \texttt{insr-paper}.
  \item[rtl] Switches the default language direction for RTL workflows.
  \item[python] Loads \texttt{pyluatex} if available.
  \item[minted] Loads \texttt{minted} if available and should be used with shell escape enabled.
  \item[review] Loads \texttt{todonotes} if available and enables colored review-note commands.
  \item[externalize] Enables TikZ externalization for large PGFPlots or TikZ workloads.
\end{description}

\section{Clinical Review Helpers}

\begin{itemize}
  \item Use \texttt{\textbackslash INSRTodoClinical} for clinical review items.
  \item Use \texttt{\textbackslash INSRTodoBiostats} for statistical-review items.
  \item Use \texttt{\textbackslash INSRTodoTech} for technical-review items.
\end{itemize}
