\chapter{Package Overview}
The INSR LaTeX package provides a complete LuaLaTeX-first document system for interdisciplinary trauma research. It includes shared style definitions, scientific-paper, presentation and clinical-manual classes, Overleaf-compatible build settings and GitHub CI integration.
\chapter{Class Options}
\begin{description}
\item[blindreview] Masks author and institution metadata where supported.
\item[twocolumn] Enables a compact two-column article layout for `insr-paper`.
\item[rtl] Switches the default language direction for RTL workflows.
\item[python] Loads `pyluatex` if available.
\item[minted] Loads `minted` if available and should be used with shell escape enabled.
\item[review] Loads `todonotes` if available and enables colored review-note commands.
\item[externalize] Enables TikZ externalization for large PGFPlots/TikZ workloads.
\end{description}
