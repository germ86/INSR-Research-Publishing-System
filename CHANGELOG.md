# Changelog

## Unreleased

- Migrated the public entry model toward INSR v4.0 with `insr.cls`, stable `main.tex`, and `config/project-config.tex`.
- Added two-phase class bootstrap, document-type adapters, document profiles, palette/theme/typography engines, semantic scientific components, focused examples, and CI/static validation for the new foundation.

## 0.1.0 - 2026-07-14

- Initial complete INSR LaTeX package scaffold.
- Added TDS-style package files under `tex/latex/insr`.
- Added paper, Beamer and clinical manual classes.
- Added Overleaf/GitHub-ready `latexmkrc`, examples, CI and package documentation.
- Added optional `python`, `minted`, `review` and `externalize` feature flags.
